# Career Copilot AI

AI-powered job application assistant built with React, FastAPI, PostgreSQL and local LLMs.

## Planned Features

- Job analyzer
- Resume matcher
- AI interview coach
- AI cover letter generator
- Application tracker
- AI recommendations

## Tech Stack

Frontend: React + TypeScript + Vite

Backend: FastAPI

Database: PostgreSQL

AI: Ollama

Deployment: Docker (later)
